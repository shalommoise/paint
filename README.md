# Paint Clone

A simple application that mimics some of the features of microsoft paint.

## How to use

User is able to:

- draw with different colours
- change background colour
- Rub the drawing
- invert backgorund and pen colour
- change pen size
- change canvas size
- restart the drawing
- change colour that has already been drawn
